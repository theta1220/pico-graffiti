# Himmo（ハイモ）

## Himmoとは
### ゲームボーイ/ファミコンのような、ピコピコサウンドをPC上で再現できる作曲ソフト
制作楽曲サンプル
https://soundcloud.com/jyct8fymajfz/wmskkcs88oqt <br>

## Himmoの仕様
### 特徴
* トロンボーンやバイオリンのような「音の垣根がない」音の表現が可能です

### 音
- 9音まで同時に再生
- 矩形波、三角波などを使用できます
- 動的に演算して波形を生成し、音を流しています

### 操作
- 液タブ上での使用を想定
- ドラッグアンドドロップでメロディを打ち込み
    - Gキーでグリッドに沿わせることが可能
- 0-9の数字キーで打ち込むトラックを切り替え
- Shift + ドラッグで移動
- Spaceで再生
- Shift + Space で最初から再生
